Walkthrough

Forensics / Steganography

Description

We were given a PNG image of the "File File Crocodile". The challenge description hinted that the crocodile had swallowed a locked archive and that the file signatures had been partially digested. It also mentioned that the only word the crocodile knew was "croc".

Enumeration

First, identify the file type:

file file_file_crocodile.png
Output:
PNG image data, 800 x 1280, 8-bit/color RGBA

Since the challenge suggested a hidden archive, I checked for embedded data.

binwalk file_file_crocodile.png
This revealed extra data after the PNG image.

Analysis

PNG files end with the IEND chunk. Everything after that chunk is suspicious.

Inspecting the appended data showed that the archive signature was corrupted:

FC 03 04

A normal ZIP archive begins with:

50 4B 03 04

PK 03 04

The challenge description specifically mentioned that the crocodile had "digested the file signatures", suggesting that the ZIP header had been intentionally modified.

Extraction

Extract everything after the PNG end marker and repair the ZIP signature.

from pathlib import Path
data = Path("file_file_crocodile.png").read_bytes()
iend = data.index(b"IEND") + 8
archive = data[iend:]
archive = archive.replace(b"FC", b"PK")
Path("croc.zip").write_bytes(archive)

Now try extracting the archive.

unzip croc.zip

The archive requested a password.

Password Discovery

The challenge stated:

Interrogating him didn't work as the only word he seemed to know was "croc".

Using that as the password:

unzip -P croc croc.zip

Reading the extracted flag file revealed:

boroCTF{n3v3r_sm1l3_4t_4_p0lygl0t_cr0c0d1l3}

proof locker

Evidence

Clue 01

PNG IEND marked the end of the image and start of appended data.

Clue 02

The ZIP header was corrupted from PK 03 04 to FC 03 04.

Clue 03

The challenge text hinted that the password was “croc”.

after action report

Lessons Learned

  • Always inspect bytes after a PNG IEND chunk.
  • Broken magic bytes can be repaired when the intended format is clear.
  • Challenge flavor text often contains the password or key.
