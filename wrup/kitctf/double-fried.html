Challenge 3: Double-fried (Network Forensics)

Challenge Description

A kitchen_log.pcap file containing syslog messages. The flag is transmitted character by character.

Analysis

Using tshark to extract all syslog messages:

tshark -r kitchen_log.pcap -Y "syslog" -T fields -e _ws.col.Info

The output shows messages R0016 through R0067 containing the flag characters, with interleaved Fxxxx messages.

Extracting the Flag

# Extract only the single-character messages
tshark -r kitchen_log.pcap -Y 'syslog.msg matches "^[A-Za-z0-9{} _,]$"' -T fields -e syslog.msg | tr -d '\n'

Final Flag

GPNCTF{NICe, y0U fOuND oU7 wH0 DID noT b3LONG 7Her3}

after action report

Lessons Learned

  • Forensics challenges often hide data in ordering, not encryption.
  • Protocol-specific filters reduce noise quickly.
  • Single-character messages should be checked for ordered reconstruction.
