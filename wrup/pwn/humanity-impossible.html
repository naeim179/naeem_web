Humanity Impossible

Pwn / Scripting

Description

عند الاتصال بالسيرفر:

nc tnkemaq46125.boroctf.com 56354

كان يعرض في كل مرة رقمًا عشوائيًا بالهيكس:

Please enter 0x174 characters!

وإذا أرسلنا عددًا أقل:

Too short!

وإذا أرسلنا عددًا أكبر:

Too long!

إذًا المطلوب هو إرسال عدد الأحرف المطلوب بالضبط.

Analysis

القيمة تتغير في كل مرة:

0x174

0x37

0x41

...

لذلك لا يمكن استخدام قيمة ثابتة.

نحتاج إلى:

قراءة الرقم من السيرفر.

تحويله من Hex إلى Decimal.

إرسال نفس العدد من الأحرف.

تكرار العملية حتى نحصل على الفلاج.

Exploit Script

from pwn import *
import re
io = remote("tnkemaq46125.boroctf.com", 56354)
while True:

line = io.recvline().decode()

print(line.strip())
m = re.search(r'0x([0-9a-fA-F]+)', line)
if not m:

break

length = int(m.group(1), 16)
io.sendline(b"A" * length)
print(io.recvall().decode())

Takeaway

التحدي لم يحتج إلى استغلال ثغرة، بل كان يعتمد على أتمتة التفاعل مع السيرفر وقراءة القيم الديناميكية وتحويلها ثم إرسال المدخلات المناسبة بسرعة ودقة. هذا النوع من التحديات يختبر مهارات الـ scripting واستخدام أدوات مثل Pwntools أكثر من اختبار ثغرات الذاكرة التقليدية.

Lessons Learned

  • Automating dynamic prompts is a core CTF skill.
