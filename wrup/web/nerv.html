Evidence

Clue 01

The query parameter is rendered by the server and reflected in magi-result.

Clue 02

Jinja object traversal reached os.popen().

Clue 03

find / -name flag* confirmed the flag path was /flag.txt.

after action report

Lessons Learned

  • When a guessed path fails, enumerate instead of assuming.
  • SSTI can become file read or RCE if dangerous globals are reachable.
  • Keep grep filters narrow so large command output remains readable.
