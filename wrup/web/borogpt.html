proof locker

Evidence

Clue 01

The render endpoint accepts a JSON template field.

Clue 02

X-Dev-Mode and the bearer token allow access to the debug render path.

Clue 03

The SSTI payload reaches os.popen() and reads /flag.txt.

after action report

Lessons Learned

  • Template render APIs need strict sandboxing.
  • Debug/dev-mode paths can expose dangerous functionality.
  • Object traversal payloads are a common SSTI escalation path.
